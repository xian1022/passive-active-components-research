01

一、研究摘要

1. 核心結論

被動元件與主動元件是所有電子系統的基礎。主動元件負責運算、控制、放大、開關與訊號處理；被動元件負責儲能、濾波、穩壓、阻抗匹配與保護；石英晶體與 Timing 元件則提供系統時脈與同步基準。

AI server、電動車、資料中心與光通訊升級，正在同時推動三條產業主軸：

  1. 高功率化

    2. GPU／ASIC 功耗提高、EV 800V、高瓦數 PSU，使 MOSFET、SiC、GaN、PMIC、MLCC、電感、薄膜電容需求提高。

  1. 高頻寬化

    2. AI cluster 對 800G／1.6T 光通訊需求增加，推動光模組、DSP、TIA、Driver、雷射、PD、Switch ASIC 與高速 SerDes。

  1. 高同步化

    2. 高速訊號需要低 jitter、低相位雜訊與高穩定時脈，推動石英晶體、XO、TCXO、VCXO、OCXO、Clock IC 與 MEMS oscillator。

二、產業總覽

1. 被動元件是什麼？

被動元件通常不需要外部電源即可發揮基本電性功能，本身不會主動放大訊號，也不會進行邏輯運算。它主要負責讓電路穩定、安全、可控。

橫向捲動查看完整表格→
元件 功能 簡短解釋
電容 儲能、濾波、去耦、穩壓 暫時存放電荷，讓電壓不要劇烈波動
電感 儲能、濾波、穩定電流 暫時存放磁能，常用於 DC-DC 轉換
電阻 限流、分壓、電流偵測 控制電流大小，協助監測與保護
磁性元件 電源轉換、濾波、隔離 用磁場傳遞或儲存能量
變壓器 升降壓、隔離 在 PSU、OBC、工業電源中重要
石英晶體 頻率基準 提供穩定時脈參考
SAW／BAW 濾波器 RF 濾波 選擇特定頻段，常見於手機與通訊設備

2. 主動元件是什麼？

主動元件需要外部電源才能運作，能控制電壓或電流，因此可以放大訊號、切換訊號、控制電路或執行運算。

MCU 控制與低功耗運算 車用、工業、家電、BMS
類比 IC 訊號放大、轉換、監控 工業、車用、資料中心
PMIC 電源管理、電壓調節 GPU board、AI server、手機、車用
MOSFET 中低壓高速開關 PSU、DC-DC、主板電源、48V server
IGBT 高壓大功率開關 工業馬達、EV inverter、太陽能逆變器
SiC MOSFET 高壓、高效率、高耐溫 EV 800V、快充、資料中心 PSU

光電元件 光電轉換 光模組、CPO、資料中心互連
RF 元件 高頻收發與濾波 手機、Wi-Fi、基地台、車用通訊

4. 主動元件為什麼能控制、放大、開關訊號？

主動元件的核心是電晶體。電晶體可以利用小訊號控制大電流，因此可以放大訊號；也可以在導通與截止之間切換，因此可以作為電子開關。大量電晶體整合後，就形成 CPU、GPU、ASIC、MCU、PMIC 等 IC。

橫向捲動查看完整表格→
功能 對應原理 代表元件
控制 用電壓或電流控制另一段電路 MCU、PMIC、Driver IC
放大 小訊號控制大電流 類比 IC、RF PA
開關 在導通與截止之間切換 MOSFET、IGBT、SiC、GaN
運算 大量電晶體執行邏輯運算 CPU、GPU、ASIC
訊號處理 將訊號轉換、調整或同步 ADC、DAC、DSP、Clock IC

5. 被動元件為什麼負責儲能、濾波、穩壓、阻抗匹配與保護？

被動元件不會主動控制訊號，但可以透過本身的電阻、電容、電感特性，穩定電壓、電流與訊號品質。

橫向捲動查看完整表格→

儲能 電容、電感 暫存電能或磁能
濾波 電容、電感、磁珠 去除雜訊與紋波
穩壓 MLCC、鋁電容、固態電容 平滑電源輸出
阻抗匹配 電阻、電容、電感 讓高速訊號傳輸更穩定
保護 電阻、TVS、保險絲、磁性元件 限流、防突波、防 EMI
隔離 變壓器、光耦、磁性元件 提高安全與抗干擾能力

6. 電子系統中三者如何搭配？

以 AI server GPU board 為例：

AI GPU／ASIC
→ PMIC 控制多組電源軌
→ MOSFET 進行高速開關
→ 電感完成 DC-DC 降壓與儲能
→ MLCC 去耦、濾波、抑制電壓波動
→ 電阻進行電流偵測與保護
→ 石英晶體／XO 提供時脈同步
→ PSU、變壓器、磁性元件供應穩定電力
→ 形成完整 AI server 系統

石英晶體與振盪器是電子產品的「時脈心臟」。一般 XTAL 偏向被動元件，XO／TCXO／VCXO／OCXO 則因內含振盪電路，通常可視為 Timing 模組或頻率控制元件。

3. 石英元件為什麼受惠 AI 光通訊？

AI cluster 需要大量高速交換器與光模組。光模組從 400G 升級到 800G／1.6T 後，對 reference clock 的 jitter 要求提高。速度越高，時脈抖動越容易造成 bit error，因此低 jitter XO、VCXO、TCXO 與 Clock IC 重要性提高。
